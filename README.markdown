README SUMMARY
===========================

Hifi-Shop is an ecommerce site for the audiophile community, specializing in vintage audio equipment.

The application is built with Spree - a complete open source commerce solution for Ruby on Rails.

Refer to the [Spree ecommerce project page](http://spreecommerce.com) to learn more.
